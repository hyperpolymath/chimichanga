= Chimichanga
:toc: auto

A capability attenuation framework for sandboxed WASM execution in Elixir.

== Overview

Chimichanga provides secure WASM execution with fine-grained capability control in Elixir applications.

== Features

* Capability-based security
* WASM sandboxing
* Elixir integration
